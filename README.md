## Project Overview

This project is a dual-format clock application written in C++. The program allows users to input a starting time in 24-hour format and displays in both:

- 12-Hour Format (AM/PM)
- 24-Hour Military Format

Users can increment the time by one hour, minute, or second using a menu to select from number-associated buttons. This program serves to compare the time formats and convert military time into a 12 hour format with ease.
